# Installation

## Stable release

To install genisys, run this command in your terminal:

```sh
uv add genisys
```

Or if you prefer to use `pip`:

```sh
pip install genisys
```

## From source

The source files for genisys can be downloaded from the [Github repo](https://github.com/InvictusLab/genisys).

You can either clone the public repository:

```sh
git clone https://github.com/InvictusLab/genisys
```

Or download the [tarball](https://github.com/InvictusLab/genisys/tarball/main):

```sh
curl -OJL https://github.com/InvictusLab/genisys/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd genisys
uv sync
```
